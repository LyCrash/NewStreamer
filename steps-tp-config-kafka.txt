1- Installation
(through docker, check kafka website)

docker pull apache/kafka:4.1.1
docker run -p 9092:9092 --name kafka apache/kafka:4.1.1 

2- Create a topic:
docker exec -it kafka /opt/kafka/bin/kafka-topics.sh --create --topic temperature --bootstrap-server localhost:9092 

3- Manual Testing:
- Produce:
docker exec -it kafka /opt/kafka/bin/kafka-console-producer.sh --topic temperature --bootstrap-server localhost:9092

- Consume:
docker exec -it kafka /opt/kafka/bin/kafka-console-consumer.sh --topic temperature --from-beginning --bootstrap-server localhost:9092
